# Portfolio WOW Design Spec

## Concept: "Digital Cosmos" - Interactive Space Voyage

### Core Experience
- Hero: Full-screen 3D particle universe that responds to mouse. Click anywhere = particle explosion bloom effect
- Scroll reveals sections like traveling through space
- Minimal text, maximum visual impact
- Every element has a purpose and interaction

### Golden Ratio Typography
- Body: 16px
- Sub-heading: 16 × 1.618 = 26px
- Heading: 26 × 1.618 = 42px
- Hero: 42 × 1.618 = 68px
- Line height: 1.618 × font size

### Color Palette (Blue cosmos)
- Void: #010108
- Deep: #050520
- Surface: rgba(8, 12, 40, 0.6)
- Primary: #4f8ffa (electric blue)
- Accent: #a78bfa (lavender)
- Glow: #7dd3fc (sky)
- Star: #fbbf24 (gold for stars)
- Text: #e8edf5
- Muted: #7b8ba3

### Interactive Moments
1. Hero click = particle bloom explosion that creates flowers/constellation
2. Scroll = camera moving through space
3. Project cards = 3D flip/morph on hover
4. Skills = orbital ring animation
5. Contact = magnetic buttons
6. Easter egg: Konami code = fireworks
